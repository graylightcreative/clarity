API ENDPOINTS (40+)
───────────────────────────────────────────────────────────────────────────────

AUTHENTICATION
POST /auth/register              - Create user account
POST /auth/login                 - User login (JWT)
POST /auth/logout                - Logout (token invalidation)
POST /auth/refresh               - Refresh JWT token
POST /auth/forgot-password       - Request password reset
POST /auth/reset-password        - Reset password with token

USER MANAGEMENT
GET  /user/profile               - Get user profile
PUT  /user/profile               - Update user profile

LICENSING
POST /license/activate           - Activate license (online)
POST /license/deactivate         - Deactivate machine
GET  /license/status             - Check license validity
GET  /license/list               - List user's licenses
POST /license/generate           - [ADMIN] Create license

PURCHASES
POST /purchase/create-checkout   - Initiate Stripe checkout
GET  /purchase/history           - User's purchase history
POST /purchase/webhook           - Stripe webhook handler

ANALYTICS
POST /analytics/telemetry        - Plugin sends usage data
GET  /analytics/dashboard        - [ADMIN] View analytics

SUPPORT
POST /support/ticket             - Submit support ticket
GET  /support/tickets            - List user's tickets

DOWNLOADS
GET  /downloads/latest           - Get latest plugin version
GET  /downloads/versions         - List available versions
